## 2.1 Modelagem Formal dos Problemas (Spec 2.1)

O projeto aborda dois problemas de otimiza√ß√£o distintos, cada um com sua pr√≥pria modelagem matem√°tica.

### A. Rota TSP (Branch and Bound / Branch and Cut)

Este √© um Problema do Caixeiro Viajante (TSP - *Traveling Salesperson Problem*).
```
* **Vari√°veis de Decis√£o:**
    * $x_{ij} = 1$ se a rota vai diretamente do ponto $i$ ao ponto $j$.
    * $x_{ij} = 0$ caso contr√°rio.
    * $u_i$ = Vari√°vel auxiliar inteira para elimina√ß√£o de sub-rota (usada na formula√ß√£o PuLP/MTZ).

* **Fun√ß√£o Objetivo (Minimiza√ß√£o):**
    * Minimizar a dist√¢ncia total percorrida. Onde $d_{ij}$ √© a dist√¢ncia (custo) entre $i$ e $j$.
    * $$\min Z = \sum_{i=0}^{N-1} \sum_{j=0}^{N-1} d_{ij} \cdot x_{ij}$$

* **Restri√ß√µes:**
    1.  **Sair de cada ponto 1x:** $\sum_{j \neq i} x_{ij} = 1, \forall i$ (Cada ponto deve ter exatamente uma sa√≠da).
    2.  **Chegar em cada ponto 1x:** $\sum_{i \neq j} x_{ij} = 1, \forall j$ (Cada ponto deve ter exatamente uma chegada).
    3.  **Elimina√ß√£o de Sub-rotas (MTZ):** $u_i - u_j + N \cdot x_{ij} \le N - 1, \forall i,j > 0, i \neq j$ (Garante um *tour* √∫nico e conectado).
    4.  **Integridade:** $x_{ij} \in \{0, 1\}$ (As decis√µes s√£o bin√°rias).

### B. Rota por Or√ßamento (Heur√≠stica Gulosa)

Este √© um Problema de Coleta de Pr√™mios (*Prize-Collecting Problem*).

* **Vari√°veis de Decis√£o:**
    * $y_i = 1$ se o ponto $i$ √© visitado.
    * $y_i = 0$ caso contr√°rio.

* **Fun√ß√£o Objetivo (Maximiza√ß√£o):**
    * Maximizar o *score* de popularidade total dos pontos visitados. Onde $p_i$ √© a popularidade do ponto $i$.
    * $$\max Z = \sum_{i=1}^{N} p_i \cdot y_i$$
    *(Nota: A heur√≠stica implementada usa um *score* ponderado (popularidade / tempo) para tomar decis√µes gulosas.)*

* **Restri√ß√µes:**
    1.  **Or√ßamento de Custo:** $\sum_{i=1}^{N} c_i \cdot y_i \le C_{max}$ (O custo total de entrada $c_i$ n√£o pode exceder o custo m√°ximo $C_{max}$).
    2.  **Or√ßamento de Tempo:** $\sum_{i=1}^{N} (t_{visita,i} + t_{desloc,i}) \cdot y_i \le T_{max}$ (O tempo total (visita + deslocamento) n√£o pode exceder o tempo m√°ximo $T_{max}$).
    3.  **Integridade:** $y_i \in \{0, 1\}$ (A visita √© bin√°ria).
```
